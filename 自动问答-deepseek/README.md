# DeepSeek自动对话助手插件

这是一个Chrome浏览器插件，用于自动化执行DeepSeek对话操作，支持自动提问、获取回答和参考资料。

## 功能特性

- ✅ 自动点击"新对话"按钮
- ✅ 自动输入预设问题
- ✅ 自动发送并等待回复
- ✅ 自动提取AI回答内容
- ✅ 自动获取参考资料（引用源）
- ✅ 支持多轮循环执行
- ✅ 支持JSON/Excel格式导出
- ✅ 实时状态反馈

## 安装步骤

1. 创建一个新的文件夹，将以上所有文件放入其中
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 在右上角启用"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择你创建的文件夹
6. 插件安装完成！

## 使用方法

1. **登录DeepSeek**：在浏览器中访问 `https://chat.deepseek.com/` 并登录你的账号
2. **打开插件**：点击Chrome工具栏上的插件图标
3. **输入问题**：在弹出窗口中输入问题列表（每行一个）
4. **设置循环**：设置循环次数（默认为1次）
5. **开始执行**：点击"开始执行"按钮
6. **等待完成**：插件会自动执行所有对话
7. **导出数据**：完成后点击"下载JSON"或"下载Excel"保存结果

## 注意事项

- ✅ 插件需要你在DeepSeek页面保持登录状态
- ✅ 执行期间请不要关闭DeepSeek标签页
- ✅ 如果页面加载较慢，可以适当延长等待时间
- ✅ 参考资料提取可能会因页面结构变化而失效，需要时请联系我更新选择器
- ✅ 建议设置合理的循环间隔，避免请求过快

## 技术说明

本插件基于你提供的Selenium脚本改造，使用原生JavaScript实现，主要适配了以下页面元素：

- **新对话按钮**：`a[data-testid="new-conversation"]`
- **输入框**：`textarea[data-testid="chat-input"]`
- **发送按钮**：`button[data-testid="send-button"]`
- **回复内容**：`div[data-testid="assistant-message"]`
- **参考资料**：`button[data-testid="reference-button"]` 和 `a[class^="SearchResultItem"]`

如果DeepSeek更新页面导致插件失效，请检查这些选择器是否匹配。
