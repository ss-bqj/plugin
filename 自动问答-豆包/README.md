# 自动对话助手插件

这是一个Chrome浏览器插件，用于自动化执行循环点击和输入操作。

## 功能特性

- ✅ 自动点击"开启新对话"按钮
- ✅ 自动输入预设文本内容
- ✅ 自动点击发送按钮
- ✅ 支持多轮循环执行
- ✅ 实时状态反馈

## 安装步骤

1. 下载项目文件到本地文件夹
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 在右上角启用"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 插件安装完成！

## 使用方法

1. 在目标网页上点击插件图标
2. 在弹出窗口中输入对话内容列表（每行一个）
3. 设置循环次数
4. 点击"开始执行"
5. 插件会自动执行点击和输入操作

## 注意事项

- 请确保在目标页面刷新后再使用插件
- 如果执行出错，请检查页面元素是否匹配
- 建议设置适当的循环间隔，避免请求过快
- 插件会根据页面加载情况自动等待

## 自定义配置

如需修改选择器或等待时间，请编辑 `content.js` 文件中的相关配置：
- `clickNewChatButton()` - 修改"开启新对话"按钮选择器
- `inputText()` - 修改输入框选择器
- `clickSendButton()` - 修改发送按钮选择器
- `sleep()` 调用 - 调整等待时间
